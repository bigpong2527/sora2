📌 从这里开始:

1️⃣ 打开终端并执行:

    cd /Users/pengwu/Downloads/sora
    npm run dev

2️⃣ 浏览器会自动打开应用 (http://localhost:3000)

3️⃣ 获取API密钥:
    - 访问: https://dashboard.tu-zi.com
    - 复制你的 API Key (格式: sk-xxx...)

4️⃣ 在应用中配置:
    - 点击右上角 "配置" 按钮
    - 粘贴 API 密钥
    - 点击 "保存"

5️⃣ 开始生成视频:
    - 输入提示词 (中文或英文)
    - 可选: 上传参考图
    - 选择模型 (Sora-2 或 Sora-2 Pro)
    - 点击 "生成视频"

🎉 就这样！视频会在 1-10 分钟内生成完成。
